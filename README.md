# Hungary-25 Predictor

Initial project scaffold for exploring predictive models related to the Hungarian 25 dataset.

## Project Structure

```
.
├── data/
│   ├── raw/
│   └── processed/
├── notebooks/
├── src/
│   ├── data/
│   ├── features/
│   ├── models/
│   └── simulation/
├── scripts/
├── models/
├── docs/
├── README.md
├── requirements.txt
└── .gitignore
```

This layout provides separate spaces for data, source code, experiments, and documentation.
